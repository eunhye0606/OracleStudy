○관리자 계정 관련 기능 구현
-①
- 여러명의 교수자를 사전에 등록하기 (교수자 이름, 주민번호)
- 교수자 명단 출력하기 (교수자명, 배정된과목명, 과목기간, 교재명, 강의실
			,강의진행여부)
- 교수자 데이터 수정, 삭제하기
-②
- 과정 등록하기(과정데이터 입력하기
		,과정명, 과정기간, 강의실 정보)
- 과정 데이터 출력하기(과정명, 강의실, 과목명, 과목기간, 교재명, 교수자명)
- 과정 데이터 수정, 삭제
-③
- 과목 등록하기(과목데이터 입력
		,과정명, 과목명, 과목기간, 교재명, 교수자명)
- 교수자는 미리 배정, 변경, 안해도됨
- 과목 정보 출력하기(과정명, 과목명,과목기간,교재명,교수자명)
- 과목 데이터 수정,삭제
-④ 과정당 학생 등록하기(학생명, 주민번호)
-학생정보출력하기(학생명, 과정명, 수강과목, 수강과목 총점,중도탈락사실)
-학생데이터 수정, 삭제

○교수자 계정 관리 기능 구현
- 성적 입력하기(과목에 대해 출결,실기, 필기배점 입력, 총점은 100점)
○성적 입력 기능 구현
- 수강한학생명(자동입력),개개인의 성적만 입력
- 중도탈락학생은 성적 입력화면에 안뜸
- 출결, 실기, 필기 점수 입력
- 합이 자동으로 총점에 기재
- 저장명령 --> 커밋?
- 성적정보 수정, 삭제 가능
○ 성적 출력 기능 구현
- 과목명, 과목기간, 교재명, 학생명, 출결, 실기, 필기, 총점, 등수
- 수강한 학생의 모든 정보 출력
- 중도탈락도 이미 수강한 과목에선 출력가능
- 중도탈락 사실 기재
